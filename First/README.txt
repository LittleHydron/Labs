This project contains realisation (Clip.java) and usage (Main.java) of class 'Clip'.
To run it, you have to follow next steps:
1) Download this folder;
2) Open console;
3) Compile Clip.java and Main.java files using following commands:
	javac \src\ua\lviv\*.java
4) Run main file using:
	java \src\ua\lviv\Main
